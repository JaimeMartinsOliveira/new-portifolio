LlamaScrapeAsync

Este projeto é uma prova de conceito (PoC) que integra um modelo de linguagem local da Hugging Face com ferramentas de automação de navegador fornecidas pela biblioteca Playwright, tudo envolvido em uma interface amigável com o NiceGUI. O objetivo principal é permitir a um agente de linguagem natural controlar um navegador de forma autônoma para buscar informações, realizar raspagens e interagir com páginas da web, com uma interface web simples para fins educacionais e de diversão pessoal.

Easy-Finance: Finanças Inteligentes para Autônomos

App de finanças pessoais focado em autônomos e pequenos negócios. Desenvolvi o back-end com Python (FastAPI) e o front-end multiplataforma com Ionic/React. A solução resolve a falta de clareza de caixa ao unificar transações bancárias (via API Open Finance da Belvo) com registros manuais. O diferencial é um assistente com IA que gera análises e metas financeiras, aplicando tecnologia de ponta para resolver um problema real.
